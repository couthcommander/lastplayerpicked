## Installation

```s
install.packages('devtools')
library('devtools')
install_github("lastplayerpicked", "couthcommander")
```

## Usage

```s
library(lastplayerpicked)
?league
l <- league()
ls <- league(source='steamer14')
lo <- league(source='oliver14')
lz <- league(source='zips14')
```
